# OhShift

OhShift is a smart shift scheduling platform for modern teams. We are currently building this project step-by-step.

## Built With

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4 + shadcn/ui
- Supabase

## Setup

```bash
bun install
bun run dev
```

## Features Currently Implemented

- Company Registration and Login
- Employee Onboarding via Magic Links (Resend)
- Real-time Schedule Sync
- Employee Dashboard & Shift Views
- Role-based Access (Owner/Manager vs Employee)

*More features coming soon...*
